5.1: document.querySelector() and .innerText

To edit HTML with JavaScript, we need to tell our code which element to change.
We can do this using document.querySelector()

Inside of the parentheses, we put the element’s class OR it’s id attribute.
(id is similar to class, but should be unique per element)

So, if our element has the class "test-class-1", we would write:
document.querySelector(".test-class-1")
Notice we had to use a period before the class name!

So, if our element has the id "test-id-1", we would write:
document.querySelector("#test-id-1")
Notice we had to use a hashtag/pound sign before the class name!

To prove that we’ve selected the right element, let’s change something about it.
We can use .innerText to change the text inside of an element.

5.4: adding/removing classes with .classList

We can also add, remove, and toggle classes via JavaScript.
Toggling a class means adding OR removing a class if the element already has it or not.

We can do this using classList on an element.

Our options are:

  • classList.remove("class-name")
  • classList.add("class-name")
  • classList.toggle("class-name")

This is incredibly useful for hiding/showing elements, like a menu!
