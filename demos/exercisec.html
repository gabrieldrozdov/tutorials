EXERCISE C: a bit more generative art

Let’s try adding some randomness to HTML and CSS again, but this time make it automatic!

Here the steps we’ll take:

  1. Assign an array of HTML elements to a variable using document.querySelectorAll()
  2. Create a function that picks a random element from the array to edit
  3. Edit that element’s CSS using template literals
  4. Assign a setInterval() loop to run that function repeatedly

Things to try:

  • How would you change this to go in order instead of being random?
  • What other CSS properties could be fun to edit?
