7.1: while loops

A while loops infinitely while a condition is true.
Once the condition is false, it stops looping.

while (condition) {
// your javascript goes here!
}

7.2: for loops

A for loop is just like a while loop, but contains more information about how it runs.
There are three parameters for a for loop:

  1. a number variable that starts at a specific value
  2. a condition for the loop to end at
  3. an equation for the loop to change by each time it loops

for (var variableName = 0; variableName < 10; variableName = variableName + 1) {
// your javascript goes here!
}
