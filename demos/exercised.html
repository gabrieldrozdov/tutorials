EXERCISE D: let the computer code the page

Let’s generate some HTML using loops, and then use setInterval() to animate those new divs.

Here the steps we’ll take:

  1. Create a CSS class for a container element with position relative (so that all absolutely positioned children will stay inside of it)
  2. Set up some CSS for the children of that container element
  3. Use a for loop to create a whole bunch of elements inside of a pre-existing element.
  4. Use document.querySelectorAll() to store all of these new elements inside an array.
  5. Use setInterval() to change the CSS of these elements over time.

Things to try:

  • What other CSS properties could be fun to randomize at the start?
  • What other CSS properties could be fun to play with in our setInterval()?
  • Are there any ways we could make this interactive?
