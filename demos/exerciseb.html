EXERCISE B: a bit of generative art

Let’s try adding some randomness to HTML and CSS!

Here the steps we’ll take:

  1. Assign an HTML element to a variable using document.querySelector()
  2. Create a few arrays of possible values for the element’s text and CSS
  3. Create a function that picks a random value from each of these arrays and applies it to the element
  4. Also, the function uses random math to change other CSS properties
  5. Trigger that function via an onclick attribute on a button or something
