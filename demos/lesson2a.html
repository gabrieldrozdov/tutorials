LESSON 2A: inspect panel, console.log()

Let’s see how to fix our code if something is wrong.

2A.1: the inspect panel

This lesson assumes you’re using Google Chrome.
If you aren’t, then switch to it now!

Previously, we saw that we can read data using the alert() function.
But sometimes our code can have errors.
To see errors, we need to open up the inspect panel.

Mac Shortcut: Command + Option + C
Windows Shortcut: Command + Option + C
Or, you can right-click anywhere on the page and press "Inspect".

In the inspect panel, navigate to the console tab.
You’ll notice an error message.
That’s because I created a variable but didn’t write its value in quotes.
Try fixing my code and seeing if the error goes away!

NOTE: the errors you see on this website might look a little funky because of the built-in text editor. When you’re editing a file directly, you might experience slightly different error information.
