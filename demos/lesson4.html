LESSON 4: Math.random()

4.1: getting a random number

We can easily get a random number using the Math.random() function.
This outputs a decimal number between 0 and 1 (not including 1)

4.2: range with an upper limit

To get a random number between 0 and some higher number, we multiply Math.random() by the upper limit (non-inclusive)



If we don’t want decimals, we can round our number to the nearest integer using Math.round().
This can round up or down, so now our range is 0 to 100

4.3: random entry in an array

We can use the previous approach to get a random entry in an array.
The plan is to get the number for a random index, and then use that to access the data stored at that index.

Example: var myArray = ['hi', 'hello', 'whatsup'];
So, myArray.length == 3 and Math.random()*myArray.length gives us a random number between 0 and 3.
But, Math.round(2.9) == 3, and myArray[3] is invalid since array[2] actually access the third value.
So, we always want to round DOWN instead.
We can do that using Math.floor().

Here’s our new equation for a random index:
Math.floor(Math.random()*arrayName.length)

And here is track actually accessing the array:
myArray[Math.floor(Math.random()*arrayName.length)]


4.4: range that starts above 0

Let’s say we want a random number between 20 and 80.
There are 60 (80-20 = 60) total whole numbers in that range.
So, first we multiply by the total possible numbers:
Math.random()*60

Then, we need to add the minimum possible value:
Math.random()*60+20

4.5: range with a lower limit

Let’s say we want a random number between -50 and 100
In that range, there are 150 total numbers
So the first thing we do is multiply by the total amount of possible values:
Math.random()*150

Then, we subtract by how far into the negative we’re starting at, which is -50 in this case: Math.random()*150-50
